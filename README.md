## VPS/Locally deploy!
```console
 $ sudo apt-get update -y
 $ sudo apt-get upgrade
 $ sudo apt install python3-venv
 $ git clone https://github.com/hitokizzy/Geez-Pyro
 $ cd Geez-Pyro
 $ python3 -m venv venv
 $ source venv/bin/act*
 $ pip3 install -U-r req*
 $ cp sample.env .env
```
