## VPS/Locally deploy!
```console
 $ sudo apt-get update -y
 $ sudo apt-get upgrade
 $ git clone https://github.com/hitokizzy/Geez-Pyro
 $ cd Geez-Pyro
 $ pip3 install -U-r requirements.txt
 $ cp sample.env .env
```

<h3 align="center">
   Edit <b>.env</b> with your own values and Run Bot
</h3>

```console
 $ screen -S Geez
 $ bash startgeez
```
